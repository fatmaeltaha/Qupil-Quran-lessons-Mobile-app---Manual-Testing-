# Qupil|Quran-lessons[Mobile app-Manual Testing]

## About The Project
Qupil is a mobile application designed to help users learn and understand the  Quran interactively and simply.

The application allows users to:
- Learn Quran with teachers
- Review Quran lessons anytime
- Read and understand the Quran
- Practice lessons through exercises
- Edit personal information
- Add and manage family members
  
## Download the app:
[click here](https://play.google.com/store/apps/details?id=com.qupil)

#  Manual Testing

Manual testing was performed to ensure that  application features, such as user information management, work correctly and meet the expected requirements.

## Tasks
- Wrote user stories and acceptance criteria for "My Profile" screen and "Personal Information" screen on Jira
- Created valid and invalid testcases on Zephyr
- Create a generic mobile testcases
- Executed Test Cycles
- Performed Manual Testing
- Created Bug Reports on Jira
